# Colección 01

Exploración personal sobre masa, especia y textura.

Proyecto web estático que presenta una primera colección de cookies de autor.
Producción limitada. Comunicación en privado.

---

## Estructura

- `index.html` — Página principal
- `style.css` — Estilos
- `assets/img/cookies/` — Imágenes optimizadas en WebP

---

## Desarrollo

Proyecto estático sin framework.

Para visualizar en local:

Abrir `index.html` directamente en el navegador  
o usar una extensión tipo Live Server en VS Code.

---

## Notas

Las imágenes han sido optimizadas con Node + Sharp.
No se incluyen originales en el repositorio.

Pequeñas series · sin prisa · con intención.
